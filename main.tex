\begin{abstract}
    This study presents the design and implementation of a fully automated, multi-agent workflow for comprehensive corporate due diligence and market analysis.
    Using an event-driven orchestration architecture, the system integrates Large Language Models (LLMs) with retrieval-augmented generation (RAG) to synthesize fragmented, dynamic, and unstructured web data into investment-grade intelligence.
    The pipeline dynamically queries search APIs for real-time sector and competitor data, while incorporating a specialized data extraction module that accesses hidden endpoints of the Greek Business Registry ($\Gamma$.E.MH.).
    By combining targeted API requests with OCR capabilities, the system programmatically retrieves and summarizes official financial statements and corporate modification records.
    To mitigate language model hallucinations and ensure data integrity, the architecture employs a structural fallback mechanism, defaulting to alternative open-source financial intelligence when official registry data is unavailable.
    Overall, this agentic pipeline demonstrates how modular AI workflows can bridge the gap between unstructured web signals and structured financial registries, significantly accelerating the due diligence process and providing actionable, verifiable insights for investment committees.
\end{abstract}

\textbf{Keywords:} AI agents; workflow automation; due diligence; retrieval-augmented generation (RAG); financial intelligence; large language models
